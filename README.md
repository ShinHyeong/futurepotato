#### IT를 좋아하는 사람들을 대상으로 한 SNS
- 인스타그램 클론 프로젝트

> **_기획 배경_**  
> 1. 첫 협업 프로젝트 <br/>
> 2. 짧은 구현 기간 <br/>
> 3. 익숙한 어플리케이션​ <br/>

> **_서비스 주요 기능_**  
> - 게시물 CRUD
> -  24시간 스토리 조회
> - 좋아요
> - 팔로우, 차단
